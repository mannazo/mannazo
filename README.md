## 프로젝트 목적 및 목표
목적 : 여행객을 위한 현지인 친구 매칭 서비스

why? 여행다닐 때 현지 가이드가 있으면 좋겠지만 가이드가 정해놓은 대로 움직이고 싶은게 아니라 내 목적에 맞게 움직이고 싶다.

목표 : 사용자가 여행가는 지역에서 내 목적에 맞는 로컬 가이드를 쉽고 안전하게 만나기.
